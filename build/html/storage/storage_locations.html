Storage locations

NIRD

The BCPU team has an allocated storage space on NIRD under /projects/NS9039K/. This is the main storage area for the team, and contains internally and externally produced data. The data structure of this space can be found here.

If you need to and cannot currently access this space, please contact our internal support

Tip

To see the project quota and how much space is currently being used, use the command:

dusage -p ns9039k

For information about NIRD, please refer to the Sigma2 documentation.
